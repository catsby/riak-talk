### Using Riak
## RESTful API

- Uses normal **HTTP Responce codes**
- **Reading**  
  `GET /riak/bucket/key`
- **Create** new object  
  `PUT /riak/bucket/key (user defined key)`  
  `POST /riak/bucket (random key)`
- **Delete**  
  `DELETE /riak/bucket/key`
